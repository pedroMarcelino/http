HTTP

O que é HTTP

HTTP é um protocolo que permite a busca de recursos, como documentos HTML. É a base de qualquer troca de dados na Web e é um protocolo cliente-servidor, o que significa que as solicitações são iniciadas pelo destinatário, geralmente o navegador da Web. Um documento completo é reconstruído a partir dos diferentes sub-documentos buscados, por exemplo, texto, descrição do layout, imagens, vídeos, scripts e muito mais
